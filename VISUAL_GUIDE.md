### Desktop (lg+):
- 3-column grid (2-col main + 1-col sidebar)

- Large project icon (16x16)
- Spacious padding (p-8)

### Tablet (md):

- Medium padding (p-6)

- Abbreviated labels

- Compact padding (p-4)
- Small icons
- Tab labels may wrap
- Sidebar moves below main content

## Key Interactions

1. **Tab Switching**: Click tabs to switch between Overview, Files, Comments
2. **File Upload**: Drag files or click Browse Files button
3. **Comment Reply**: Click Reply button to open nested reply form
4. **View Files**: Click View button to open file in new tab
5. **Edit Project**: Click Edit Project button in header
6. **Back Navigation**: Click Back to Projects arrow button

## Performance Notes

- Components are optimized with React hooks
- File uploads happen via API routes (no page reload)
- Comments post via API (no page reload)
- Tab switching is instant (client-side state)
- All images use next/image for optimization
- Gradients use CSS (no images needed)
