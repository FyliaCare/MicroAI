## â“ Need Help?

### Can't find a client?
- Click "Create New Client" button
- Or check if client exists in database

### Quote not saving?
- Check internet connection
- Look for "Last saved" timestamp
- Check browser console for errors

### PDF not generating?
- Try downloading again
- Check if all required fields are filled
- Contact support if issue persists

### Client can't access quote?
- Verify quote status is "sent" (not "draft")
- Check expiration date
- Ensure correct link was shared
